# Scaled SVD Components from OECD Labor Force Participation Data

An object of class `"bage_ssvd"` holding components extracted from labor
force participation data from the [OECD Data
Explorer](https://data-explorer.oecd.org). `LFP` holds 5 components.

## Usage

``` r
LFP
```

## Format

Object of class `"bage_ssvd"`.

## Source

Derived from data in the "Labor Force Indicators" table of the OECD Data
Explorer. Code to create `LFS` is in folder 'data-raw/ssvd_lfp' in the
source code for the bage package.

## Details

**Versions:**

- `"v2025"` data downloaded on 2025-10-17
